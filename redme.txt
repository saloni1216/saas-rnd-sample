django-admin : help to creating and managing Django projects, apps, database migrations, and admin tasks.

Noted: 

Main use cases
create project: django-admin startproject projectname
create app: django-admin startapp appname
database schema:
django-admin makemigrations
django-admin migrate
run checks: django-admin check
create superuser: django-admin createsuperuser
collect static for deploy: django-admin collectstatic
run tests: django-admin test
launch dev server (usually from project dir): python manage.py runserver (django-admin runserver also works with --settings)
key note
Use python manage.py <command> from project root for your specific project settings automatically (recommended).

django-admin  startproject cfehome
python manage.py runserver
